# Team Website - Ishaan & Abhiram

A modern team website showcasing the collaboration between Ishaan Saraswat and Abhiram Voleti, featuring their work on BruinLM.

## How to Run Locally

### Prerequisites
- Node.js (version 18 or higher)
- npm (comes with Node.js)

### Steps

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Start the development server**
   ```bash
   npm run dev
   ```

3. **Open your browser**
   - Navigate to [http://localhost:3000](http://localhost:3000)
   - The website will automatically reload when you make changes

That's it! The website should now be running locally.

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build the project for production
- `npm run start` - Start the production server (requires build first)
- `npm run lint` - Run ESLint to check for code issues

## Tech Stack

- **Next.js 16** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Radix UI** - Accessible component primitives
- **Lucide React** - Icons

## Project Structure

- `app/` - Next.js app directory with pages and layouts
- `components/` - Reusable React components
- `public/` - Static assets (images, icons)
- `lib/` - Utility functions
